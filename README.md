<p>توصيه هايي جهت دستيابي به سلامت 1-وزن خودتون رو در محدوده ي متعادل نگه داريد 2-به هر بهانه اي فعاليت بدني داشته باشيد 3-ت=رژيم غذايي خودتون رو تنظيم و اصلاح کنيد</p>
